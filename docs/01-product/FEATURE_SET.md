# Feature Set

## MVP Features

### 1. Family Member Directory
- list seluruh anggota keluarga
- search by name / nickname
- filter by family branch
- sort by generation / family line / recent update

### 2. Member Profile
- foto profil
- nama lengkap
- nama panggilan
- gender
- tanggal lahir
- tanggal wafat
- bio singkat
- family branch
- orang tua
- pasangan
- anak-anak
- saudara kandung (derived)

### 3. Relationship Management
- tambah orang tua dari halaman profil
- tambah pasangan dari halaman profil
- tambah anak dari halaman profil
- tambah saudara via parent linkage
- hapus relasi dengan konfirmasi jelas
- merge duplicate member

### 4. Profile Photo Management
- upload foto profil
- preview sebelum save
- replace existing photo
- crop/compress ringan client-side if needed
- storage in Supabase bucket

### 5. Search and Discovery
- search cepat by name / nickname
- hasil dengan foto profil kecil
- quick jump to profile
- filter branch keluarga

### 6. Tree View (Simple)
- fokus pada satu orang
- tampilkan parent / spouse / children
- pan/zoom basic
- tombol reset view
- mobile-friendly controls

### 7. Roles
- viewer
- contributor
- editor
- admin

### 8. Audit and Safety
- perubahan penting tercatat
- archive instead of permanent delete
- clear undo flow where possible

## Phase 2 Features
- cerita / memories per orang
- album keluarga
- timeline keluarga
- event penting
- suggested edits workflow
- family branch hub pages
- print / export as PDF or booklet
- QR code ke profil anggota

## Phase 3 Ideas
- voice notes kenangan keluarga
- video clips pendek
- relationship timeline
- auto-generated family highlights
- anniversary / birthday reminders
- family reunion page
