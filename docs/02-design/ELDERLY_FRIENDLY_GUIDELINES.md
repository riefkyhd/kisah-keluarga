# Elderly-Friendly Guidelines

## Design Goal
Membuat UI yang tetap nyaman untuk pengguna yang:
- tidak terlalu akrab dengan teknologi
- lebih sensitif terhadap tampilan padat
- membutuhkan teks yang jelas dan interaksi sederhana

## Visual Rules
- font size minimum nyaman dibaca
- line spacing lega
- kontras tinggi
- tombol besar
- icon hanya sebagai pelengkap, bukan satu-satunya penanda
- hindari elemen terlalu kecil / rapat

## Interaction Rules
- satu aksi utama yang jelas di tiap layar
- label tombol eksplisit: "Tambah Anak", bukan hanya "+"
- hindari hidden gestures penting
- modal jangan terlalu kompleks
- tampilkan feedback sukses / gagal yang jelas

## Language Rules
Gunakan bahasa yang:
- sederhana
- hangat
- familiar
- tidak terlalu teknis

Contoh:
- gunakan "Orang Tua" bukan "Ascendants"
- gunakan "Pasangan" bukan "Partner Relationship Node"
- gunakan "Anak" bukan "Descendants"

## Navigation Rules
- menu sedikit
- breadcrumb sederhana bila perlu
- tombol kembali jelas
- jangan buat user harus mengingat banyak langkah

## Recommended Feature Support
- mode font besar (optional)
- reduced motion
- high contrast mode (optional)
- clear success confirmation
- clear empty states
