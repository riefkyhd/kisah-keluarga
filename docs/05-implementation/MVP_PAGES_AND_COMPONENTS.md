# MVP Pages and Components

## Main Pages

### 1. Home Page
Purpose:
- welcome user
- search entry
- quick access to family directory
- featured stories / latest updates

Suggested components:
- HeroIntro
- SearchBar
- QuickLinks
- RecentUpdates
- FeaturedStoryCards

### 2. Family Directory Page
Purpose:
- list and search all people

Suggested components:
- SearchInput
- BranchFilter
- MemberCardGrid
- SortDropdown
- EmptyState

### 3. Member Profile Page
Purpose:
- central place to understand one person

Suggested components:
- ProfileHeader
- ProfilePhotoCard
- BasicInfoCard
- ParentsSection
- SpouseSection
- ChildrenSection
- SiblingsSection
- QuickEditActions
- StorySnippetList

### 4. Add / Edit Member Page or Modal
Purpose:
- create or update one member quickly

Suggested components:
- MemberForm
- PhotoUploader
- RelationshipQuickAttach
- SaveActionBar

### 5. Tree View Page / Section
Purpose:
- show family relationships visually

Suggested components:
- TreeCanvas
- FocusPersonSelector
- ZoomControls
- RelationshipLegend

### 6. Stories Page
Purpose:
- list family stories and memories

Suggested components:
- StoryCard
- StoryFilter
- StoryComposer (later if allowed)

### 7. Admin Dashboard
Purpose:
- operations hub for editors/admins

Suggested components:
- AdminOverview
- PendingChanges
- RecentAuditLogs
- ArchivedMembersTable
- DuplicateCandidates

## Key Reusable Components
- AvatarUploader
- ConfirmDialog

- MobileBottomActionBar
