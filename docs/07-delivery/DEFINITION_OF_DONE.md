# Definition of Done

## Feature-Level Definition of Done
A task is done only if:
- implementation works end-to-end
- role access is correct
- loading / error / empty states exist
- mobile layout works
- code is understandable
- no obvious security leak exists
- documentation is updated if needed

## MVP Definition of Done
MVP is done when:
- users can log in
- admin/editor can create and edit members
- members can have profile photos
- relationships can be created and removed safely
- users can search and view members
- family tree basic view works
- app is usable on mobile
- app is deployed on Vercel and connected to Supabase
