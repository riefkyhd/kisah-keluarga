# Project Summary

## Product Name
**Kisah Keluarga**

## Product Type
A private, web-based **family tree + family archive** application for large families.

## Main Goal
Membuat satu tempat digital yang memudahkan keluarga untuk:
- melihat struktur keluarga
- mencari anggota keluarga
- memahami hubungan antaranggota
- menyimpan foto profil tiap anggota
- menyimpan cerita, kenangan, dan momen penting keluarga

## Primary Audience
1. **Keluarga besar umum**
2. **Orang tua / lansia** yang perlu UI sederhana
3. **Admin keluarga** yang bertugas merapikan dan menjaga data
4. **Kontributor keluarga** yang ingin membantu menambahkan informasi

## Core Promise
"Menyimpan cerita, silsilah, dan kenangan keluarga dalam satu tempat yang hangat, sederhana, dan mudah dipakai."

## Why This Product Exists
Platform family tree umum sering terasa:
- terlalu rumit
- terlalu teknis
- terlalu fokus pada genealogical data formal
- kurang cocok untuk keluarga yang hanya ingin cara sederhana untuk melihat hubungan keluarga dan menyimpan kenangan

Kisah Keluarga ingin menjadi versi yang:
- lebih ramah orang Indonesia
- lebih ramah lansia
- lebih fokus ke pengalaman keluarga sehari-hari
- tidak hanya pohon keluarga, tetapi juga arsip cerita keluarga

## Product Strategy
Bukan membuat "software silsilah formal" dulu.

Strategi utama:
- mulai dari **directory anggota keluarga + profil + relasi inti**
- tambahkan **tree view**
- tambahkan **stories / memories / timeline**
- jadikan produk terasa seperti **rumah digital keluarga**, bukan hanya diagram hubungan

## MVP Success Criteria
MVP dianggap berhasil jika:
- admin bisa menambah / edit / hapus anggota keluarga dengan mudah
- tiap anggota bisa punya foto profil
- user bisa mencari nama dengan cepat
- user bisa memahami relasi inti: orang tua, pasangan, anak, saudara
- UI tetap nyaman dipakai dari HP
- keluarga bisa membuka aplikasi dari link tanpa kebingungan
